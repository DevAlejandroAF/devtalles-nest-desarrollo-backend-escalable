# Teslo API

1. Clonar proyecto

2. Instalar dependencias: ```pnpm install```

3. Clonar ```.env.template```, rellenar variables de entorno y renombrar a ```.env```

4. Levantar base de datos

```
docker-compose up -d
```

5. Ejecutar ```Seed``` de la palicaci√≥n

```
http://localhost:3000/api/seed
```

5. Levantar el proyecto: ```pnpm run start:dev```
